Create a concise summary of today from the daily note.

## Steps

1. Read `/home/maxime/NOTES/CLAUDE.md` for:
   - Daily note location
   - Summary format preferences

2. Find today's daily note

3. Read and analyze:
   - Completed tasks (checked items)
   - Events and meetings
   - Notes and decisions
   - Commits logged
   - Time tracked (if present)

4. Generate daily summary:
   - Quick overview (1-2 sentences)
   - Key accomplishments
   - Time spent (if tracked)
   - Important notes or decisions
   - Incomplete tasks (carry-over items)

5. Present summary or append to daily note under `## Summary` section

Use at end of day for reflection. Can compare against morning `/obsidian:today` list to see what was completed.
